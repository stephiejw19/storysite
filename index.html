How betterBOT Works

betterBOT uses continuous webscraping from a cloud server to update an Azure database with near-real time player, team, match, and odds data. A machine learning model, running in a separate server instance queries the database to generate betting recommendations.

This web interface allows viewers to see upcoming matches and associated odds, along with the bot recommendations. It also lets users interact with the bot, inputing their favorite teams against foes to see which the bot predicts will win.
